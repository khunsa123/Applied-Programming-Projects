# 🚗 Car Racing Game

## 📌 Overview

A simple **2D car racing game** built with Python and the **Pygame** library. The objective is to avoid collisions with an oncoming enemy car while accumulating the highest score possible. The game progressively increases in speed, making it more challenging over time.

---

## 🎮 Gameplay

- Dodge the enemy car by switching between lanes
- The game speed increases every **5,000 points**
- A collision ends the game

### 🎯 Controls

| Key | Action |
|-----|--------|
| `A` or `←` | Move left |
| `D` or `→` | Move right |

---

## 🧠 Game Rules

- Your car starts in the right lane
- The enemy car appears in the left lane and moves downward
- Colliding with the enemy car ends the game

---

## 🛠️ Tech Stack

- **Language:** Python 3.x
- **Library:** Pygame

---

## ⚙️ Installation & Usage

```bash
# Install Pygame
pip install pygame

# Run the game
python main.py
```
