# 🚗 Car Racing Game

## 📌 Overview
Welcome to the **Car Racing Game**!  
This is a simple **2D car racing game** developed using the **Pygame** library.

Your objective is to **avoid collisions with the enemy car** and reach the highest score possible as the game progressively becomes more challenging.

---

## 🎮 How to Play
- Use the **'A' key** or **Left Arrow (←)** to move your car to the left lane  
- Use the **'D' key** or **Right Arrow (→)** to move your car to the right lane  
- Avoid colliding with the enemy car  
- The game speed increases every **5000 points**, making it more challenging over time  

---

## 🚀 Getting Started

### ✅ Prerequisites
- Python 3.x  
- Pygame library  

### 📦 Installation
Install Pygame using pip:

```bash
pip install pygame

## ▶️ Run the Game
1. Download or clone the repository  
2. Navigate to the project folder  
3. Run the Python script:

```bash
python main.py

4. Use the Controls below to play

---

## 🧠 Game Rules
- Your car starts in the right lane  
- The enemy car appears in the left lane and moves downward  
- If your car collides with the enemy car, the game ends  

---

## 🎯 Controls
- **Move Left:** A or ← (Left Arrow)  
- **Move Right:** D or → (Right Arrow)  

---
