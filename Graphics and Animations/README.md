# 🎨 Graphics and Animations

## 📌 Overview

A collection of **Python-based graphics and animation scripts** built using Matplotlib. Each script demonstrates programmatic generation of animated geometric shapes, showcasing core Python animation techniques using `matplotlib.animation`.

---

## 📂 Scripts

| File | Description |
|------|-------------|
| `circle_animation.py` | Animated circle — demonstrates smooth geometric motion and rendering |
| `square_animation.py` | Animated square — demonstrates frame-based shape animation and transformation |

---

## 🛠️ Tech Stack

- **Language:** Python 3.x
- **Library:** Matplotlib (`matplotlib.animation`)

---

## ⚙️ Installation & Usage

```bash
# Install dependencies
pip install matplotlib

# Run animations
python circle_animation.py
python square_animation.py
```
