# 📐 SE Diagrams

## 📌 Overview

A collection of **software engineering diagrams and planning visualisations** built programmatically using Python and Jupyter Notebooks. This folder serves as a reference library of reusable diagram templates for project planning, scheduling, and software design documentation.

---

## 📂 Contents

| File | Description |
|------|-------------|
| `Gantt Chart Designs.ipynb` | Gantt chart templates for project scheduling and timeline visualisation |

---

## 🛠️ Tech Stack

- **Language:** Python 3.x
- **Environment:** Jupyter Notebook
- **Libraries:** Matplotlib, Pandas

---

## ⚙️ Installation & Usage

```bash
# Install dependencies
pip install matplotlib pandas jupyter

# Launch Jupyter
jupyter notebook
```

Open the relevant notebook and run all cells to render the diagrams.
